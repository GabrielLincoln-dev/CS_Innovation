# CS_Innovation

Landing page desenvolvida para a consultoria de Marketing CS Innovation, o site foca em apresentar os nichos atendidos e vantagens em contratar os serviços, possui também um formulário para contato e avaliação de negócio para qualificação de leads.


## Tecnologias utilizadas: 

HTML

CSS

Javascript

Bootstrap
